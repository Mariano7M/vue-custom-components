# vue-components-lib

## Install

```
npm install @mnho/vue-components-lib
```

## Basic Usage

```
<template>
  <SecondButton></SecondButton>
</template>

<script>
import SecondButton from './components/SecondButton.vue'

export default {
  name: 'App',
  components: {
    SecondButton
  }
}
</script>

```
